VGC队伍构建检查清单 来自MysticVGC的X（推特）帖子：

我做了一个检查清单，可以帮助那些想入门VGC并进行队伍构建的新手。我被问过几次这个问题，所以我列出了我组队时会遵循的步骤。把VGC想象成国际象棋，因为你正在制定策略来击败你的对手。

在队伍构建时，你通常是已经知道想用什么宝可梦，还是会拿一个环境主流队伍（meta team）然后将其改造成自己的风格？你是否尝试从零开始组建队伍却不知从何入手？这份清单可能会对从零开始组建队伍有所帮助！

- 你想要一个环境主流队伍还是一个非传统队伍？

- 你的队伍结构是什么？
	- 速攻手 (Sweeper)
	- 坦克/肉盾 (Tank)
	- 辅助 (Support)
	- 重装辅助 (Bulky Support)
	- 脆皮速攻手 (Frail Sweeper)
 
- 在队伍构建时，你是否已经有或想要一个核心？
	- 火/水/草 (重火力输出核心)
	- 龙/妖精/钢 (奇幻核心/钢仙龙）
	你可以尝试构建一个核心并看看效果如何：
	- 电/地面/飞行
	- 超能/恶/幽灵
	- 电/地面/龙

- 你想加入某种战术模式吗？
	- 雨天 (Rain)
	- 晴天 (Sun)
	- 顺风 (Tailwind)
	- 戏法空间 (Trick Room)
	- 顺风戏法空间 (TailRoom - 即队伍中同时有顺风和戏法空间的选项)

- 如何在队伍中找到战斗模式：
	- 在某些对局中，你是否总是感觉必须派出同样的4只宝可梦？
	- 你的队伍中是否有在战斗中发现彼此配合特别好的宝可梦？
	- 你的队伍中是否有感觉需要其他队员配合才能发挥好，否则就不好用的宝可梦？
	- 在选择首发后，你是否常常清楚后排哪两只宝可梦配合最佳？

- 你的队伍意图是什么？了解你决策背后的原因！
	- 例如，你为什么决定给毽子棉 (Jumpluff) 带上气势披带 (Focus Sash)？
	- 例如：加入炽焰咆哮虎 (Incineroar) 或风速狗 (Arcanine)，因为它们是优秀的威吓手 (Intimidators)。给败露球菇 (Amoonguss) 0点速度个体值 (IVs)，使其在戏法空间下成为（相对）最快的宝可梦（即最慢出手）。测试那些适合内敛 (Modest) 性格的宝可梦，看看带生命宝珠 (Life Orb) 还是讲究眼镜 (Choice Specs) 最优。给戟脊龙 (Baxcalibur) 分配196点攻击努力值 (EVs) 并携带冰柱坠击 (Icicle Crash)，以确保一击必杀 (OHKO) 一只0 HP / 0防御的烈咬陆鲨 (Garchomp)。
	- 不要思虑过度。你只是需要理解你队伍构建时所做决策背后的逻辑。

- 你想围绕哪只宝可梦或哪种机制进行构建？你对什么感兴趣？
	- 例如：班基拉斯 (Tyranitar) 的沙暴队 (Sand Team) 来造成逐步伤害，超能强攻队 (PsySpam) 来造成极高的超能力属性伤害，吃吼霸 (Dondozo) + 米立龙 (Tatsugiri) 的组合来获得巨大的耐久和吃吼霸的伤害能力。

- 你向队伍中添加宝可梦的意图是什么？
	- 使用最爱？
	- 热门/强大？
	- 你是否非常惧怕某个特定的宝可梦或属性？
	- 它是否有良好的协同性？
		- 例如：如果你的队伍中有卡璞・鳍鳍 (Tapu Fini) 和纸御剑 (Kartana)，加入风速狗 (Arcanine) 可以覆盖它们的弱点。如果你有猛攻形态的武道熊师 (Rapid-Strike Urshifu) 太晶化 (Tera) 为水属性以造成更高的水属性伤害，你可以加入一个避雷针 (Lightning Rod) 特性的宝可梦来吸引电属性招式。
	- 在某只宝可梦身上使用特定道具来激活它。
		- 例如：使用跺脚 (Stomping Tantrum) 来触发巨金怪 (Metagross) 的弱点保险 (Weakness Policy)。
		- 例如：给具有毅力 (Guts) 特性的月月熊 (Ursaluna) 带上火焰宝珠 (Flame Orb)，因为毅力特性在宝可梦处于异常状态时会提升50%的攻击力。
	- 为什么那只宝可梦应该在你的队伍里？ 如果某个进攻型宝可梦出现在60%的队伍中，而你有一只宝可梦能够抵抗其强力攻击并能做出反击，那么这只宝可梦就很适合你的队伍。

- 思考在对战中哪些宝可梦总是出现。那么你应该做出什么调整？

- 它在你队伍中的属性是否良好？
	- 能否轮换上场抵挡威胁场上宝可梦的强力攻击，例如炽焰咆哮虎 (Incineroar) 在使用抛下狠话 (Parting Shot) 后换下，之后可以再次换上场对敌方宝可梦施加更多威吓(Intimidate)。
	- 它能否承受攻击并掌控战局？
	- 通过利用防御属性和进攻打击面来给对手施加压力。
	- 把VGC想象成国际象棋。你的宝可梦能否迫使对手撤退，例如迫使他们换人？

- 这些宝可梦在进攻端如何互补？
	- 完成击杀。
	- 造成互补的伤害。
	- 为彼此创造更多攻击机会。

- 这些宝可梦在防守端如何互补？
	- 为彼此进行联防换人。
	- 弥补彼此的弱点。

- 你的宝可梦们协同性好吗？
	- 例如：天气手 (Weather Setter) + 天气打手 (Weather Sweeper)。
	- 例如：击掌奇袭 (Fake Out) + 戏法空间 (Trick Room)。
	- 例如：戏法空间手 (Trick Room Setter) + 慢速攻击手 (Slow Attackers)。
	- 例如：威吓 (Intimidate) + 脆皮速攻手 (Frail Sweepers)。
	- 例如：慢节奏宝可梦 (Slow Pace Pokémon) + 喜欢强化的宝可梦 (Pokémon that like to Set Up)。
	- 例如：看我嘛 (Follow Me) + 喜欢强化的宝可梦。
	- 例如：看我嘛 (Follow Me) + 戏法空间 (Trick Room)。
	- 例如：守住 (Protect) + 戏法空间 (Trick Room)。
	- 例如：击掌奇袭 (Fake Out)/守住 (Protect) + 顺风 (Tailwind)。

		- 戏法空间 (Trick Room)
	- 重新导向 (Redirection)
		- 愤怒粉 (Rage Powder)
		- 看我嘛 (Follow Me）
	- 免疫粉末类招式 (Immune to Powder Moves)
		- 草属性 (Grass Types）
		- 防尘护目镜 (Safety Goggles)
		- 防尘特性 (Overcoat)
	- 开墙 (Screens)
		- 反射壁 (Reflect)
		- 光墙 (Light Screen)

		- 广域破坏/魂舞烈音爆 (Breaking Swipe) (降低攻击1级)
		- 冰冻之风 (Icy Wind) (降低速度1级)
		- 电网 (Electro-Web) (降低速度1级)
	- 强化招式 (Set Up Moves)

		- 腹鼓 (Belly Drum)
		- 诡计 (Nasty Plot)
		- 蝶舞 (Quiver Dance)

	- 回复招式 (Recovery Moves)

- 队伍构建时宝可梦的配置框架
	- 通用配置
		- 本系招式1 (STAB 1)
		- 本系招式2 (STAB 2)
		- 功能性/打击面招式(Utility/Coverage)
		- 守住 (Protect)
	- 辅助配置
		- 本系招式 (STAB)
		- 降低能力值招式 (Stat Drop)
		- 重新导向招式 (Redirection)
		- 回复/速度控制招式 (Recovery/Speed Control)
	- 强化配置
		- 本系招式1 (STAB 1)
		- 本系招式2 (STAB 2)
		- 强化招式 (Set Up Move)
		- 守住 (Protect)
	- 极限进攻配置 (Hyper Offensive Outline)
		- 本系招式1 (STAB 1)
		- 本系招式2 (STAB 2)
		- 本系招式3 (STAB 3)
		- 本系招式4 (STAB 4)
	- 纯辅助配置 (Full Support)*
		- 异常状态类招式 (Status Condition mode)
		- 回复类招式 (Recovery mode)
		- 重新导向招式 (Redirection)
		- 守住 (Protect)
		*使用此配置时需注意，挑衅 (Taunt) 招式将是这类宝可梦用法的天敌。
